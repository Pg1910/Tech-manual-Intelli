# Tech-manual-Intelli
technical-manual and intelligence mainaintance doctrine with Q &amp;A system using RAG corpus pipeline.

### Current problem and challenge faced by militaries across the world inclusively for the indian military .

In the present times the indian military owns a number of different arsenal all accross the world and with varied use cases and installment dependencies all over India. To over come this challenge of manual and doctrines that needs to repeatedly followed and tracked by the idnian army with the minute current sota modifications or removal of the datasheets .. Here in this project ,I am developing a RAG pipeline that will ingest manual /doctrines from open source , summarise the context with a LLM integration alogn with the question answer prompting . I will be also training a small LLM head of sub billion params to showcase flexiblity of reducing  the reliablity on only open source models ..giving chunking inhouse based outputs which will be verifiable , citaton -pinned , refuse-when-unsure-QA over technical manuals.
